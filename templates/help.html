Tech stack: HTML, CSS, JavaScript, Python

We used HTML and CSS to create the web interfaces, and for functions like the search bar, we used JavaScript to provide options for the user to select. We chose to use Flask to bind the web app together.

We chose to use a content-based filtering system, over a collaborative filtering system, as it was easier for us to implement and users did not have to log in to use the website. Our algorithm thus filtered out details like directors, actors, ratings of the film of choice, and searched across the IMDB database to find similar movies based on these attributes.
